**目的：** 依頼者からの初回問い合わせを受け付け、基本情報を収集する

**URL：** `https://[事務所ドメイン]/consult`

**表示要素：**

```

│  柳田法律事務所 オンライン相談                          │
├─────────────────────────────────────────────────────────┤
│                                                         │
│  ┌─────────────────────────────────────────────────┐   │
│  │  【重要】                                        │   │
│  │  ・このチャットは受任を保証するものではありません │   │
│  │  ・法的助言は受任後に弁護士が行います           │   │
│  │  ・利益相反等の理由により受任できない場合があります│   │
│  └─────────────────────────────────────────────────┘   │
│                                                         │
│  ┌─────────────────────────────────────────────────┐   │
│  │  チャットエリア                                  │   │
│  │                                                 │   │
│  │  [AI] はじめまして。柳田法律事務所の相談受付     │   │
│  │       チャットです。借金のお悩みについて         │   │
│  │       お聞かせください。                         │   │
│  │                                                 │   │
│  │  [ユーザー] 借金が300万円あって...               │   │
│  │                                                 │   │
│  │  [AI] ご状況をお聞かせいただきありがとうございます。│   │
│  │       一般的なご案内として、債務整理には         │   │
│  │       いくつかの方法があります...               │   │
│  └─────────────────────────────────────────────────┘   │
│                                                         │
│  ┌─────────────────────────────────────────────────┐   │
│  │  入力欄                                    [送信] │   │
│  └─────────────────────────────────────────────────┘   │
│                                                         │
│  Powered by ZettAI（システム提供）                      │
└─────────────────────────────────────────────────────────┘
```

**機能：**
- チャットボットによる一次ヒアリング
- 借金総額、借入先数、収入状況などの基本情報収集
- 面談予約への誘導

**非弁対策：**
- 免責表示を常時表示
- 「あなたは破産が最適」等の個別判断は出さない
- 一般的な制度説明に限定

**AIの回答例（OK）：**
- 「一般に、債務整理には自己破産、個人再生、任意整理などの方法があります」
- 「詳しい判断は弁護士との面談で行います」

**AIの回答例（NG）：**
- ~~「あなたの場合は自己破産が最適です」~~
- ~~「この状況なら免責される可能性が高いです」~~
